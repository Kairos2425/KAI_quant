# KAI Quant - 芯片股量化分析仪表盘

> 沪深A股半导体板块7只代表性芯片股的交互式量化分析平台

## 项目简介

本项目通过 Tushare API 获取沪深股市中芯片/半导体相关公司的历史交易数据，构建了一个实时可更新的交互式量化分析网页，支持多股票对比、收益率分析、风险评估等功能。

## 功能特性

- **多股票对比**: 7只代表性芯片股（中芯国际、韦尔股份、北方华创、中微公司、紫光国微、长电科技、寒武纪）
- **归一化走势**: 以起始交易日为基准(100)的归一化价格对比
- **量化指标**: 年化收益率、年化波动率、夏普比率、最大回撤
- **风险收益分析**: 散点图展示各股票的风险收益比
- **成交量分析**: 最新交易日成交量对比
- **收益排名**: 累计收益率横向排名
- **交互式筛选**: 可切换显示/隐藏股票、切换图表类型
- **实时数据更新**: 支持通过Tushare API刷新最新交易数据

## 数据说明

- **数据源**: [Tushare Pro](https://tushare.pro)
- **时间范围**: 2025年7月1日 - 2026年6月30日（约240个交易日）
- **数据字段**: 开盘价、最高价、最低价、收盘价、涨跌幅、成交量、成交额等
- **数据格式**: CSV (chip_stocks_daily.csv) + JSON (data/stocks_data.json)

## 使用方法

### 在线访问
访问 GitHub Pages: https://kairos2425.github.io/KAI_quant/

### 本地运行
```bash
# 1. 获取最新数据
python fetch_all.py

# 2. 生成交互式仪表盘
python build_html.py

# 3. 用浏览器打开 index.html
```

## 技术栈

- **数据获取**: Python + Tushare HTTP API
- **前端可视化**: Chart.js 4.x
- **数据处理**: Python (pandas, json, csv)
- **部署**: GitHub Pages
