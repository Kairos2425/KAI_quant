## 一、海龟策略核心思想

海龟策略诞生于 1983 年 Richard Dennis 与 William Eckhardt 的一场著名赌局：他们招募一批**毫无交易经验**的学员（戏称"海龟"），仅用两周时间传授一套**完全机械的趋势跟踪规则**，随后给每人一笔资金实盘。结果多数"海龟"在 4 年里年化收益远超市场均值，证明了一件事——**优秀的交易是可以被系统化、被传授的**，而非单纯依赖天赋或直觉。

其核心思想可概括为三点：

1. **趋势跟踪（Trend Following）**：价格突破过去 N 日的高低点通道，意味着一波趋势启动，顺势入场；趋势反转则离场。
2. **波动率自适应仓位（Volatility-based Sizing）**：用 ATR 衡量波动，波动大则少买、波动小则多买，使**每一笔交易承担的风险恒定**（账户权益的 1%）。这是海龟最聪明之处。
3. **截断亏损、让利润奔跑（Cut loss, Let profit run）**：2×ATR 硬止损把单笔最大亏损锁死在约 2% 权益；盈利后金字塔加仓，让趋势利润自然放大。

理查德·丹尼斯原版给出**两套系统**：
- **系统一（S1）**：20 日突破买入、10 日跌破离场（灵敏，信号多）。
- **系统二（S2）**：55 日突破买入、20 日跌破离场（稳健，信号少）。
本实现以 S1 为默认并支持自由调参，便于观察"通道周期"这一核心杠杆的影响。

### 关键优势
- **完全机械化、去情绪化**：买卖、仓位、止损全由规则决定，杜绝"扛单""追涨杀跌"的人性弱点。
- **风险预算明确**：每单位风险 = 账户权益 × 1%，无论账户大小、标的贵贱，单笔风险恒定。
- **波动自适应**：同样一笔资金，在暴雷股上自动轻仓、在慢牛股上自动重仓，天然实现"风险平价"。
- **趋势捕获力强 + 回撤可控**：2×ATR 止损使策略在连续止损后仍有资本翻盘；金字塔让趋势利润奔跑。

## 二、关键概念详解

### 1. 高低点通道（Donchian Channel，唐奇安通道）
- **上轨（突破线）** = 过去 N 日的最高价：`upper[i] = max(high[i−N .. i−1])`
- **下轨（离场线）** = 过去 M 日的最低价：`lower[i] = min(low[i−M .. i−1])`

收盘价**向上突破上轨** → 买入信号（趋势启动）；**向下跌破下轨** → 卖出信号（趋势结束）。
通道把"趋势"翻译成可机械判断的边界，是海龟的"触发器"。

> 回测细节：信号由**前一日收盘价**计算（与前一日之前的 N 日通道比较，规避前视偏差），实际成交在**次日开盘价**执行；上轨/下轨均已 `shift(1)`，确保通道不包含参考日自身。

### 2. 平均真实波幅 ATR（Average True Range）
先算**真实波幅 TR**（刻画单日真实振动幅度，考虑了跳空）：
```
TR[i] = max( 当日最高−最低, |最高−昨收|, |最低−昨收| )
```
再对 TR 做 **Wilder 平滑**（RMA，区别于简单移动平均）：
```
ATR[N]   = mean(TR[0..N−1])                 # 种子
ATR[i]   = (ATR[i−1]·(N−1) + TR[i]) / N      # i > N
```
ATR 是海龟的"尺子"——它**同时决定买多少、在哪里止损**：
- 仓位：单位股数 =（账户权益 × 1%）/ ATR → 价格反向走 1 个 ATR，亏损刚好是账户的 1%。
- 止损：建仓价 − 2×ATR。

### 3. 基于 ATR 的头寸规模（Unit）
```
单位股数 Unit = (账户权益 × 1%) / ATR
```
含义：若价格反向运动 1 个 ATR，该单位亏损恰好为账户权益的 1%。**波动越大（ATR 越大）→ 单位越小 → 自动轻仓**；波动越小 → 单位越大。这就是"波动率自适应"。

### 4. 2×ATR 止损
建仓后设置止损价 `stop = 入场价 − 2×ATR`。若价格跌穿止损，立即离场，单笔最大亏损约 **2% 权益**。
金字塔加仓后，止损随成本上移：`stop = 入场价 − (2 − 0.5×(单位数−1))×ATR`，每加一单位上移 0.5×ATR，逐步锁定利润。

### 5. 金字塔加仓（Pyramiding）
价格每上涨 `0.5×ATR` 加 1 个单位，最多 4 个单位。让盈利头寸在趋势延续时自然变大——这是海龟"让利润奔跑"的关键机制。

## 三、量化效果基础指标
（与双均线任务一致，便于横向比较）

- **累计回报 Cumulative Return** = 期末净值 / 初始资金 − 1，反映整个回测期的绝对收益。
- **最大回撤 MDD** = `min(净值 / 历史最高点 − 1)`，最坏情况潜在亏损，风控核心指标（越接近 0 越好）。
- **夏普比率 Sharpe Ratio** =（日均超额收益 / 日均收益标准差）× √252，每单位风险换来的超额回报，>1 良好、>2 优秀。
- **年化收益 / 年化波动率**：按 252 交易日年化，便于跨周期比较。
- **买入持有对比 Buy & Hold**：同期"买了不动"的回报，用于判断策略是否真创造价值（趋势策略在强牛市常跑输买入持有）。
- **最大单位 / 交易次数**：反映仓位集中度与换手频率。

## 四、中芯国际（688981.SH）实证

在默认参数 **通道 20/10（S1）、ATR 20、成本 0.1%** 下：

| 指标 | 数值 |
|---|---|
| 累计回报 | **+8.48%** |
| 年化收益 | +9.08% |
| 最大回撤 | **−21.35%** |
| 夏普比率 | **0.414** |
| 年化波动率 | 37.54% |
| 买入持有（基准） | +80.97% |
| 交易次数 | 17 次 |
| 最大持仓单位 | 4 / 4 |

**解读**：中芯国际本区间单边上行（买入持有 +80.97%），但海龟仅录得 +8.48%，明显跑输。这并非策略"失效"，而是其**固有特征**：
- 海龟用 2×ATR 止损，在上涨途中遇到回调（哪怕只是 2 个 ATR 的正常波动）就会被洗出，随后若再突破又重新入场，来回摩擦；
- 高波动个股的 ATR 大 → 仓位偏小、止损空间大，上涨时"赚得慢"、洗盘时"割得狠"；
- 其价值不在收益最大化，而在**回撤可控、规则透明、可机械复制、可规模化**——对资金管理的意义大于对收益的意义。

### 中芯国际不同通道周期的表现（ATR=20）
| 通道(上/下) | 累计回报 | 最大回撤 | 夏普 | 交易 |
|---|---|---|---|---|
| 10 / 5 | +32.92% | −13.68% | 1.057 | 23 |
| 10 / 10 | +20.53% | −19.79% | 0.727 | 23 |
| 10 / 20 | **+41.66%** | −19.79% | **1.131** | 16 |
| 20 / 5 | +20.38% | −14.76% | 0.733 | 17 |
| 20 / 10（默认） | +8.48% | −21.35% | 0.414 | 17 |
| 20 / 20 | +30.39% | −21.35% | 0.883 | 16 |
| 55 / 5·10·20 | −14.62% | −15.76% | −0.997 | 11 |

可见：**短通道（E10）整体优于长通道（E20/E55）**；对 中芯国际 这类有清晰波段趋势的标的，E10/X20 组合最优（+41.66%，夏普 1.13）。而 55 日长通道在本区间完全失效（−14.62%），说明过长的突破线在该年份跟不上节奏。

## 六、适用场景与应用心得

### 适用场景
- **中长线单边趋势市**、商品/指数/强趋势个股的主升浪；
- 作为**机械执行、去情绪化**的趋势跟踪骨架，尤其适合需要严格纪律的资金管理。

### 不适用
- **横盘震荡市**（信号反复、摩擦成本吞噬收益，紫光国微、中微公司即此）；
- 低流动性或跳空频繁的品种（2×ATR 止损难以成交，滑点放大）。

### 心得
1. **波动自适应是精髓**：海龟最聪明之处不是"突破买入"，而是用 ATR 把仓位与波动挂钩。同样的规则，在不同波动环境自动调仓，这正是它能在 1980 年代那场实验中胜出的底层逻辑。
2. **止损是生命线**：2×ATR 把单笔亏损锁死在约 2% 权益，使策略在连续止损后仍有资本翻盘。没有止损的趋势跟踪等于裸奔。
3. **大数定律与低胜率**：海龟胜率往往仅 40% 上下，靠"截断亏损、让利润奔跑"的正期望生存——少数大赢家覆盖多数小亏损。切忌因几次止损就放弃规则。
4. **参数是情景而非真理**：默认 20/10 是经典值，但历史调参易过拟合。建议用多组参数取稳健区间（如本报告的多周期对比），并叠加成交量/基本面过滤。
5. **与双均线策略对比**：双均线用两条均线交叉判断趋势，海龟用高低点通道突破 + ATR 仓位管理。海龟在"风险管理"上更体系化（明确的单位风险与止损），但二者本质都是**趋势跟踪**，在强趋势市跑输买入持有、在震荡市被反复打脸的规律高度相似。
